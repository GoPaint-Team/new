Linux Installation Instructions

After downloading, extract the archive.

Then, move the extracted folder into your /usr/local/bin directory using:

sudo mv [Extracted Folder] /usr/local/bin/

To uninstall, simply delete the directory:

sudo rm -r /usr/local/bin/[Extracted Folder]

Note: Replace [Extracted Folder] with the name of the folder you extracted.
